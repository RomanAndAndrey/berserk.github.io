ИНСТРУКЦИЯ ПО ДОБАВЛЕНИЮ СТРАНИЦ МАНГИ
=====================================

1. Папки были созданы для вас в соответствии с начальной конфигурацией в `js/script.js`.
   Структура: assets/manga/vol{X}/ch{Y}/

2. Поместите ваши изображения в соответствующие папки.
   Наименование файлов: 01.jpg, 02.jpg, 03.jpg ... 10.jpg
   (Скрипт добавляет ноль перед числами меньше 10).

3. Чтобы добавить обложки томов:
   - assets/manga/vol1/cover.jpg
   - assets/manga/vol2/cover.jpg

4. Чтобы добавить больше Томов/Глав:
   - Откройте `js/script.js`
   - Обновите объект `mangaDatabase` новыми томами/главами/количеством страниц.
   - Создайте соответствующие папки в `assets/manga/`.
